# BukuMedia Next - Java Book Games

Proyek ini adalah platform pembelajaran interaktif berbasis web yang dibangun dengan Next.js. Aplikasi ini menyediakan serangkaian permainan edukatif (level) yang dirancang untuk membantu siswa dalam pembelajaran dan refleksi diri.

## üöÄ Teknologi

- **Framework**: [Next.js](https://nextjs.org/)
- **Bahasa**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Database**: [MongoDB](https://www.mongodb.com/)
- **Autentikasi**: [NextAuth.js](https://next-auth.js.org/)

## üì¶ Fitur Utama

- **Sistem Level**: 30 level permainan interaktif dengan berbagai jenis tantangan (Word Search, Drawing, Input Refleksi, Drag & Drop).
- **Dashboard Siswa**: Melacak kemajuan level dan pencapaian.
- **Dashboard Guru**: Memantau aktivitas siswa dan melihat jawaban detail.
- **Export Data**: Fitur export data siswa ke format Excel.
- **Responsive Design**: Tampilan yang menyesuaikan dengan berbagai perangkat.
